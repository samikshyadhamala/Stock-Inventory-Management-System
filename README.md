# Stock Management System

The Stock Management System is built using **PHP** and **MySQL**. It provides an easy-to-use interface for managing inventory operations.

The system helps businesses:

- Monitor product stock levels
- Manage suppliers
- Record purchase transactions
- Track sales
- Generate inventory reports

This project demonstrates the implementation of a **complete inventory management workflow**, including authentication, database interaction, and reporting features.

---

## Features

- User authentication (login, registration, OTP verification)
- Product and batch management
- Supplier management
- Purchase order tracking
- Sales management
- Automatic stock updates
- Report generation and Excel export
- Email verification using PHPMailer

---

## Technologies Used

- PHP  
- MySQL  
- HTML, CSS, JavaScript  
- PHPMailer
